<h2>Build</h2>
make

<h2>Run</h2>
./x86 -i haribote.img

<h2>Language and Library</h2>
C++11, SDL2.0

<h2>How to use mouse</h2>
<h3>Grab mouse on x86 emulator</h3>
click mouse on x86 emulator

<h3>Release mouse out of x86 emulator</h3>
press lef-ctrl+left-alt+g
